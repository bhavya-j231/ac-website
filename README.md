## 🚀 Getting Started

Wanna run it locally and get chillin'? Follow these steps:

```bash

git clone https://github.com/bhavya-j231/ac-website.git

# 2. Move into the directory
cd ac-website

# 3. Install dependencies
npm install

# 4. Fire it up!
node server.js
